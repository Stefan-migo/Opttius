## 🎬 Estructura de Cada Video (YouTube)

### Template de 5 minutos

| Timestamp | Sección                 | Duración |
| --------- | ----------------------- | -------- |
| 0:00-0:15 | **Hook**                | 15 seg   |
| 0:15-0:45 | **Problema/Dolor**      | 30 seg   |
| 0:45-3:30 | **Demo ( paso a paso)** | 2:45     |
| 3:30-4:30 | **Beneficio/Resultado** | 1 min    |
| 4:30-5:00 | **CTA**                 | 30 seg   |

### Hook Examples

| Tipo      | Ejemplo                                                             |
| --------- | ------------------------------------------------------------------- |
| Pregunta  | "¿Cuánto tiempo te toma crear un presupuesto de presbicia?"         |
| Métrica   | "Este proceso me tomaba 2 horas. Ahora me toma 2 minutos."          |
| Resultado | "En este video vas a aprender a crear presupuestos 10x más rápido." |
| Dolor     | "¿Te ha pasado que pierdes una receta?"                             |

## 📱 Extractos para Redes

### Cómo maximizar un video en 5 piezas de contenido

| Pieza | Plataforma          | Duración  | Desde el video               |
| ----- | ------------------- | --------- | ---------------------------- |
| 1     | YouTube (completo)  | 3-8 min   | Video original               |
| 2     | LinkedIn (post)     | -         | Texto del video + screenshot |
| 3     | LinkedIn (extracto) | 30-60 seg | Grabación pantalla           |
| 4     | Instagram Reel      | 15-30 seg | Grabación pantalla           |
| 5     | Instagram Story     | 15 seg    | Screenshots paso a paso      |

### Tips para Extractos Efectivos

1. **Graba en 16:9** para YouTube
2. **Corta en 9:16** para Reels (recadrage vertical)
3. **Texto overlay grande** - legible sin sonido
4. **Primeres 3 segundos son clave** - hook o resultado
5. ** Música lo-fi** para YouTube (sin copyright)

## 🎥 Guía de Producción

### Configuración de Grabación

| Elemento       | Recomendación                                  |
| -------------- | ---------------------------------------------- |
| **Resolución** | 1920x1080 (mínimo)                             |
| **Framerate**  | 30 fps                                         |
| **Audio**      | Mic dedicado o AirPods Pro                     |
| **Captura**    | OBS (gratis), Loom, Camtasia                   |
| **Edición**    | CapCut (móvil), Premiere Rush, DaVinci Resolve |

### Configuración de Pantalla

| Elemento           | Recomendación                         |
| ------------------ | ------------------------------------- |
| **Zoom interfaz**  | 100% (no zoom para mejor legibilidad) |
| **Cursor**         | Mostrar, sin highlight excesivo       |
| **Sidebar**        | Contraída si no es relevante          |
| **Notificaciones** | DESACTIVAR todas                      |
