## 2. Estética Obligatoria (Sistema Epoch)

| Elemento            | Especificación                                                                  |
| ------------------- | ------------------------------------------------------------------------------- |
| **Paleta**          | Verde profundo (#1A2B23), Dorado (#C5A059), Crema (#F9F7F2), Charcoal (#1A1A1A) |
| **Tipografía**      | Sans-serif geométrica (Inter, Geist, DM Sans). Sin serifas decorativas.         |
| **Formas**          | Bordes `rounded-xl` o `rounded-2xl`. Minimalismo.                               |
| **Estilo**          | Lujo tecnológico. Escaparate visual. Coherente con landing.                     |
| **Texto en imagen** | Máximo 1-2 líneas. Fuente grande, legible en móvil.                             |

## 4. Checklist Pre-Publicación

Antes de subir cada post:

- [ ] ¿El copy evita legado, forjar, santuario, maestría, pergamino, credencial?
- [ ] ¿Los beneficios están cuantificados cuando aplica (posts 1, 2)?
- [ ] ¿El link opttius.cl está en el caption?
- [ ] ¿La imagen usa paleta Epoch?
- [ ] ¿El texto en imagen es legible en móvil (si aplica)?
- [ ] ¿La estética es minimalista y coherente con la landing?
