# Guía de Usuario - Opttius

Documentación explicativa del sistema Opttius para ópticas. Esta guía está orientada al usuario final, con lenguaje simple y no técnico.

## Secciones

- [Dashboard](/admin/docs/dashboard) - Visión general y panel de control
- [Punto de Venta](/admin/docs/pos) - Ventas y caja
- [Trabajos de Laboratorio](/admin/docs/trabajos) - Órdenes de trabajo y taller
- [Presupuestos](/admin/docs/presupuestos) - Cotizaciones para clientes
- [Citas y Agenda](/admin/docs/citas) - Agendamiento de consultas
- [Productos e Inventario](/admin/docs/productos) - Catálogo y stock
- [Clientes](/admin/docs/clientes) - CRM y gestión de clientes
- [Analíticas](/admin/docs/analiticas) - Reportes y métricas

## Acceso

La documentación está disponible dentro del panel de administración en **Documentación** o desde el botón "Ver Documentación" en el Dashboard.
