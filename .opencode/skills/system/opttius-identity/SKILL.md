# Opttius Identity — Guía de Identidad de Marca

**Pivote 2026-02-22:** Lujo Tecnológico y Precisión Clínica (estilo Stripe, Apple, Vercel).

## Arquetipo Objetivo

**Software médico de alta gama / Minimalismo SaaS.** Combina autoridad clínica (creado por tecnólogo médico) con el pragmatismo y velocidad de una startup de Silicon Valley. No vendemos "software"; vendemos el motor inteligente para ópticas modernas.

## Narrativa Central

1. **Origen:** De la clínica al código. 100% nativo para ópticas.
2. **Especialización:** Diseñado desde cero para ópticas — no adaptación de CRM genérico.
3. **Valor:** Automatiza. Controla. Crece.

## Principios de Copy

### Tono de Voz

- **Autoridad clínica:** Tecnólogo médico, precisión oftalmológica.
- **Pragmatismo startup:** Beneficios concretos, velocidad, eficiencia.
- **Sin metáforas literarias:** Eliminar legado, forjar, santuario, maestría, pergamino, credencial, llave de acceso.

### Palabras Clave (usar)

- Automatiza, controla, crece, centraliza, tecnólogo médico, 100% nativo, exclusivo para ópticas, recetas, inventario, laboratorio, presupuestos, agenda.

### Palabras Prohibidas

- Legado, forjar, santuario, maestría, pergamino, credencial, llave de acceso.

### Palabras a Evitar

- Genéricos de SaaS: "revolucionario", "disruptivo", "game-changer".

## Sistema Visual (Epoch)

### Paleta (intacta)

- `epoch-primary`: #1A2B23 (verde profundo)
- `epoch-accent`: #C5A059 (dorado)
- `epoch-surface`: #1A1A1A (charcoal)
- `epoch-background`: #F9F7F2 (crema)

### Tipografía (Minimalismo SaaS)

- **Principal:** Geist, Inter o DM Sans (sans-serif geométricas)
- **Acento decorativo (mínimo):** Cormorant Garamond solo para itálicas sutiles
- **Eliminadas:** Cinzel, Playfair Display, Lato

### Formas

- **Cards y modales:** `rounded-xl` o `rounded-2xl`
- **Botones:** `rounded-xl`
- **Eliminado:** `rounded-arch` (arcos arquitectónicos)
- **Espaciado:** Generoso (p-8, p-12)

## Copy Oficial

- **Eslogan:** "Automatiza. Controla. Crece."
- **Badge:** "De la clínica al código. 100% nativo para ópticas."
- **Hero subtitle:** "Desde el examen visual hasta la entrega del lente. Centraliza tus recetas, inventario, flujos de laboratorio y ventas en una única plataforma."
- **Auth:** "Email", "Contraseña" (no credencial, llave de acceso)

## Checklist de Identidad

Antes de publicar copy o UI:

- [ ] ¿El copy evita legado, forjar, santuario, maestría, pergamino, credencial, llave de acceso?
- [ ] ¿El eslogan "Automatiza. Controla. Crece." está presente donde aplica?
- [ ] ¿Se usa tipografía sans-serif geométrica (Geist/Inter/DM Sans)?
- [ ] ¿Las formas usan rounded-xl/2xl (no rounded-arch)?
- [ ] ¿La paleta Epoch se mantiene intacta?
- [ ] ¿Los CTAs son claros ("Probar Gratis", "Iniciar sesión", "Registrarse")?
